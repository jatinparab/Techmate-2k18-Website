About the event

Techmate 2k18 is a technical festival, organised by CSI-RAIT. Every technical geek needs some moments of recreation. Here’s a complete package of fun filled events that will balance your technical see-saw.

National Level Technical Paper Presentation

Have a Research Paper waiting to be unleashed in the world of intellectuals ?? A golden opportunity awaits for you to present your work before the esteemed panel.

read more

C'Mulator

This is a general purpose programming language event . The participant's hold on 'C’ language will be tested along with their approach towards solving problem statements, of varying difficulty levels.

read more

Java Jargons

This one is the event based on Java coding which will consist of 3 coding rounds where you will have to show your coding skills. Basically this event will test your coding skills in Java language.

read more

Code Arena

This is a tournament based event wherein participants will be pitted against one another. The participant who solves the given problem in minimum time will go to next level. Participants have choice of selecting their preferred language.

read more

Placement Mantra

An event that refers to ‘mock’ placement rounds for all the undergraduate students who would be sitting for campus interviews.It tests your General Abilities , Aptitude , Knowledge , Team Building Skills,etc.

read more

Aim 4 Masters

Preparing for GRE /GATE ??


This Event tests your Reading , Listening and Logical Reasoning Skills with the aid of Mock-Aptitude test , Story Telling , Essay Writing and Blog/Article Evaluation.

read more

Gwiggle

An event based solely on the capability of how you use GOOGLE. In this event the participants will have to use the lesser known features of Google search engine. You will have to create a set of questions which can be solved using these features.

read more

Market Mayhem

Have you ever tried your hand with stocks? If you haven’t this is the perfect place to start. From getting you acquainted with how the stock market works to how carefully you have to plan your each investment so that it would turn out to be profit.

read more

Lan Gaming

Series of multiplayer games like COUNTER STRIKE , NFS , FIFA where teams compete with one another and the winning team qualifies for the next round

read more

Cric-o-Lumina

Ever imagined playing Cricket in the Dark,where you can see nothing but the ball! No Players, No Arena Only objects of importance will be lit up during gameplay

read more

Glowball

This is Dark Room Soccer Tournament. Where you can’t see a thing except football and goal post . You gotta aim for the goal. Participants will have to use their other senses to their full extent .

read more

Takeshi's Castle

Each contestant will be provided with EXCITING tasks. They have 60-seconds to complete each task using props that are provided to them. Complete the task and win points.

read more
